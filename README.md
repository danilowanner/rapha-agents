# Browser MCP LLM Agent

Minimal Node + TypeScript project demonstrating an LLM-driven agent that uses Browser MCP for controlling a logged-in Chrome and the Vercel AI SDK for model runtime. Uses `zod` for validation and `better-sqlite3` for persistence.

Quickstart

1. Copy `.env.example` to `.env` and set keys.
2. Install dependencies:

```bash
npm install
```

3. Run:

```bash
npm run start
```
